# Ollama-Subtitle-Toolkit
為小型 LLM 優化的離線字幕翻譯工具組  (翻譯、補翻、簡轉繁)
這是一套專為小型離線語言模型（如 Qwen2.5-1.5B, Llama3.2-3B）優化的字幕翻譯工具組。

## 🌟 為什麼選擇這套工具？
小型模型速度快且硬體要求低，但在翻譯長篇字幕時容易產生「漏翻」或「中英混雜」。本工具採用「三位一體」的工作流程，完美解決小模型的智力瓶頸。

## 🛠️ 工作流程
1. transrt_fast.py (高速初步翻譯)：
   - 快速將英文字幕翻譯成中文（簡繁不限）。
   - 支援 Context（上下文）參考，解決 but... 等斷句問題。
2. retranslate_en.py (深度修補)：
   - 自動掃描漏翻行及中英混雜行。
   - 針對頑固行進行二次強力翻譯，解決 assuredly tell you... 等混雜問題。
3. s2t_srt.py (純淨簡轉繁)：
   - 將翻譯結果統一轉換為繁體中文。
   - 純淨指令設計，避免 1.5B 模型產生複讀循環。

## 🚀 快速開始
1. 安裝依賴：
   pip install -r requirements.txt
2. 確保 Ollama 已執行並下載了您需要的模型（如 qwen2.5:1.5b）。
3. 依序執行三部曲，即可獲得高品質繁體字幕。
