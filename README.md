# Ollama Subtitle Toolkit

這是一套專為小型離線語言模型（如 Qwen2.5-1.5B, Llama3.2-3B）優化的字幕翻譯工具組。

## 🌟 為什麼選擇這套工具？
小型模型速度快且硬體要求低，但在翻譯長篇字幕時容易產生「漏翻」或「中英混雜」。本工具採用「三位一體」的工作流程，完美解決小模型的智力瓶頸。

## 🛠️ 工具介紹

### 1. transrt_ultra.py (終極一鍵版) 🚀
**推薦使用！** 一次執行即可自動完成所有翻譯流程：
- 自動進行初翻、漏翻掃描修補、以及簡轉繁。
- 適合電腦效能充足，希望放著讓它自動跑完的場景。
- **使用方式：** `python transrt_ultra.py`

### 2. 三部曲分開執行 (適合手動控制)
如果您想分階段確認翻譯品質，可以依序執行：
1. `transrt_fast.py` (高速初步翻譯)：
   - 快速將英文字幕翻譯成中文。
2. `retranslate_en.py` (深度修補)：
   - 自動掃描漏翻行及中英混雜行並進行二次翻譯。
3. `s2t_srt.py` (純淨簡轉繁)：
   - 將翻譯結果統一轉換為繁體中文。

## 🚀 快速開始
1. 安裝依賴：
   `pip install -r requirements.txt`
2. 確保 Ollama 已執行並下載了您需要的模型（如 `qwen2.5:1.5b`）。
3. 執行 `python transrt_ultra.py` 並依照提示輸入路徑。
